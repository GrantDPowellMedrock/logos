# MedRock & MedDots — Logo Library

Every official MedRock and MedDots logo, ready to download and use. Scroll down to see them all, or jump to one using **[Find a logo](#find-a-logo)** below.

## How to use these logos

**To download a logo:** click its picture — it opens the full **SVG** (best quality) in your browser, where you can right-click and choose **“Save image as…”**. (Each logo is previewed at 512 px here just to keep the page tidy.)

Need a specific PNG size, or a link to copy? Under each logo, click **“Other sizes & direct links”** to expand the full set of sizes plus copy-paste links (for websites, email signatures, slide decks).

**Should I use the SVG or a PNG?**

- **SVG** — the original artwork. It stays perfectly sharp at *any* size, from a business card to a billboard. Use this whenever you can.
- **PNG** — a ready-made picture at a fixed size. Use it when a tool only accepts a regular image (some email tools, older apps, website favicons).

**Which PNG size?**

| Size | Good for |
|---|---|
| 16–48 px | favicons, tiny icons, email signatures |
| 64–256 px | websites, social media, slide decks |
| 512–1024 px | print, posters, large screens |

*Not sure? Use the SVG — it never blurs.*

Every logo has a **see-through (transparent) background**, so it sits cleanly on any color.
